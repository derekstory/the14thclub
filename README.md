# The 14th Club — website

The public site for *The 14th Club* (iOS golf game): landing page, privacy policy,
and support/FAQ. Static HTML, no build step. Served via GitHub Pages at
[the14thclub.com](https://the14thclub.com).

- `/` — landing (`index.html`)
- `/privacy` — privacy policy (`privacy/index.html`)
- `/support` — support + FAQ (`support/index.html`)
